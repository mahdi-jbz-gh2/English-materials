# Unit 17 — have & have got 
---
# Mind map

```
                         HAVE
               ┌──────────┼───────────┐
               │                      │
        POSSESSION/STATE         ACTIVITIES/EXPERIENCES
   (have / have got; no -ing)     (only have; can use -ing)
      ├─ things: a car              ├─ meals: breakfast/lunch
      ├─ relations: two sisters     ├─ hygiene: a shower
      ├─ illnesses: a headache      ├─ events: a party/holiday
      └─ appointments: a meeting    └─ misc: trouble, a look, a chat
```


* **Same meaning:** *have* = *have got* for possession/relations/illnesses/appointments. 
* **No continuous** with this meaning: ✖️ *We’re having a nice room.* ✔️ *We have/We’ve got a nice room.* 
* **Past:** use **had** (usually without *got*). 





---

## Use have for actions/experiences

| Category        | Common expressions                                       | Notes                        |
| --------------- | -------------------------------------------------------- | ---------------------------- |
| Meals           | **have breakfast / lunch / dinner / a cup of coffee**    | Action meaning “eat/drink”.  |
| Hygiene         | **have a bath / a shower / a swim**                      | ✔ can use **I’m having…**.   |
| Events/Time off | **have a break / a rest / a party / a holiday**          | —                            |
| Talk/Look       | **have a chat / a discussion / a conversation / a look** | —                            |
| Problems/Fun    | **have trouble / difficulty / fun / a good time**        | Note the uncountable ones.   |
| Birth           | **have a baby**                                          | = give birth.                |

* **Have got is not possible** in these expressions. 
* Compare: ✔ *Sometimes I **have** a sandwich for lunch (eat).* ✖ *I’ve got…* (for eating). 
* For these expressions you may use **continuous**: *We’re **having** a great time.* *He’s **having** a shower.* 
* Questions/negatives here use **do/does/did**: *I **don’t** usually have a big breakfast. Where **does** Chris usually have lunch? Did you **have** trouble…?*

## Common mistakes

```
WRONG: I’m having a car.          → RIGHT: I have / I’ve got a car.  (state; no -ing) :contentReference[oaicite:18]{index=18}
```

## Quick forms table

| Time        |  Main pattern                                          | Example                                         |
| ----------- | ----------------------------------------------------- | ----------------------------------------------- |
| Present     | **have / have got**                                   | *They **have / have got** a new car.*           |
| Present Q   | **Do you have…? / Have you got…?** | *Do you have any questions?*                    |
| Present Neg | **don’t have / haven’t got**          | *I don’t have any questions.*                   |
| Past (all)  | **had**; Q/Neg with **did**                           | *Did you have a car…? I didn’t have my phone…*  |

> **Rule of thumb:** no **-ing** for possession/state.
