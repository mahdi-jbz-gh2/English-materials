# Unit 38 — if I do … and if I did …
---
## Mind map (one-screen review)

```
                  IF-CLAUSES (present/future meaning)
                         ┌───────────────┬─────────────────┐
                         │               │                 │
                REAL possibility   UNREAL / unlikely     Notes
               if + PRESENT,        if + PAST,            no WOULD in IF-part;
               main clause:         main clause:          use WOULD/COULD/MIGHT
               will/can/may         would/could/might     in the other part
```

* **Real**: *If we **take** the bus, it **will be** cheaper.* 
* **Unreal**: *If we **took** the bus, it **would be** cheaper. (We are going to take the train.)*
* **Meaning note**: **if + past** here talks about **now/future imagination**, not past time. 

---

## Decision chart

```
Are you talking about a real/likely plan?
         │
     Yes ├─► if + PRESENT  → main clause: will/can/may
         │      If it rains, we'll stay in.
     No  └─► if + PAST     → main clause: would/could/might
                If it rained, we would stay in.
```

(“If you find it, call me” = real; “If you **found** a wallet, what would you do?” = hypothetical.) 



---

## Common mistakes

> **Do not** use **would** in the if-part:
> ✅ *If we **went** by bus, it **would** be cheaper.* ✖️ *If we **would go** …* 

* ✖️ *If somebody **would point** a gun at me, I’d be scared.*
  ✅ *If somebody **pointed** a gun at me, I’d be scared.*
